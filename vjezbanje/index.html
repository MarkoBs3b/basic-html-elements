HTML & CSS Basics Summary

HTML summary

HTML (HyperText Markup Language) is the heart of every webpage. It defines the structure of the webpage and annotates the content to tell the browser what to display and provide extra meaning

  • HTML uses "elements" to describe (annotate) content
  • HTML elements typically have an opening tag,content and then closing tag
  • You can also have void elements like images
  • you can also configure elements attributes
  • THere is long list of available elements but you will gain experienceover time,no worries

Learn more about all available html elements on the MDN html element refernce.

CSS summary

CSS (Cascading Style Sheets) is a language that allows you to define style definitions for your HTML document.

  • styles are assigned via property value pairs
  • You can assing styles via the style attribute
  • to avoid code dupplication you use global stless elements instead
  • When working with css,concepts like inheritance should be understood
